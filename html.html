HTML Structure

The basic structure of a HTML page starts with the declaration of the document type and version of HTML. This is done with the <!DOCTYPE> tag with the value of "html" within it. This will declare the document to be a HTML5 document.

Following the declaration, comes the <html> tag, which is the root element of the HTML document. This tag contains the <head> and <body> tags.

The <head> tag contains the metadata of the document, such as the title of the document, the description of the document, and the keywords of the document.

The <title> tag contains the name of the document, and is displayed in the browser's tab.

The <meta> tag contains the metadata of the document, such as the charset, the viewport, and the description of the document.

The <body> tag contains the content of the document, such as the text, images, and other elements.

An example of a basic HTML document can be seen below:

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>My Website</title>
    </head>
    <body>
        <h1>My Website</h1>
        <p>Welcome to my first website</p>
    </body>
    </html>
