What is HTML?

HTML stands for HyperText Markup Language. It is used to create the structure and content of a website.

Compared to other programming languages, HTML is very simple and easy to learn.

It is used along side CSS, which is used to style the HTML elements.

HTML uses tags to mark up the content of a webpage. Most elements have opening and closing tags, however, some elements do not. These are called Void Elements.

HTML Structure

The basic structure of a HTML page starts with the declaration of the document type and version of HTML. This is done with the <!DOCTYPE> tag with the value of "html" within it. This will declare the document to be a HTML5 document.

Following the declaration, comes the <html> tag, which is the root element of the HTML document. This tag contains everything else in the document.

The <head> tag contains the metadata of the document, such as the title of the document, the description of the document.

The <title> tag contains the name of the document, and is displayed in the browser's tab.

The <meta> tag contains the metadata of the document, such as the encoding of the document, the viewport, and the description of the document.

The <body> tag contains the content of the document, such as the text, images, and other elements. This is what will generally appear on the webpage itself.

This is an example of the basic structure of a HTML page:

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>My Website</title>
    </head>
    <body>
        <h1>My Website</h1>
        <p>Welcome to my first website</p>
    </body>
    </html>
